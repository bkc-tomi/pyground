# Djangoをつかったウェブアプリ

## タイトル
pyground

## 実行方法
1. コンテナのビルド
    ```
    $ docker-compose build
    ```
2. コンテナの立ち上げ
    ```
    $ docker-compose up -d
    ```
3. ページを開く
    ブラウザで`http://localhost:8000`にアクセス
